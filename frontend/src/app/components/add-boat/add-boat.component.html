Ajouter un Bateau

Remplissez les détails ci-dessous.

Le nom du bateau est requis.
Le nom du propriétaire est requis.
La description est requise.
L'année est requise et doit être entre 1900 et l'année actuelle.
La longueur est requise et doit être supérieure à 0.
Le prix est requis et doit être un montant positif.
Le numéro d'immatriculation est requis.
